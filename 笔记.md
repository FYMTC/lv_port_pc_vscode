**`git clone https://github.com/microsoft/vcpkg`**

1. 在 vcpkg 根目录下，运行 vcpkg 引导程序命令：

   > **`bootstrap-vcpkg.bat`**

   引导程序将使用 Microsoft C/C++ 工具、库和 Windows SDK 的位置配置 vcpkg。

安装SDL2

vcpkg install sdl2
